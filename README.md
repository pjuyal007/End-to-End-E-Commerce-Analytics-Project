# SQL + Power BI E-Commerce Analytics Project
# Project Overview

This project is an end-to-end e-commerce analytics solution built using SQL and Power BI.

# The project focuses on:

customer analytics
revenue trends
product performance
customer segmentation
business KPI tracking

A star schema data model was created using fact and dimension tables to improve analytical performance.

# Tools & Technologies
SQL
Power BI
DAX
Data Modeling

Project Workflow


# 1. Data Import

Imported raw ecommerce datasets into SQL database.

# 2. Data Cleaning

Handled:

null values
incorrect datatypes
datetime conversion
duplicates
# 3. Data Modeling

Created:

fact_orders
dim_customers
dim_products
dim_sellers

Implemented star schema relationships.

# 4. SQL Analysis

Performed:

revenue analysis
monthly trends
customer segmentation
product/category analysis
top customer analysis
# 5. Power BI Dashboard

Built interactive dashboards for:

customer insights
revenue growth
product analysis
